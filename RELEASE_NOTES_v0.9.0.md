# HomeBridge v0.9.0 release notes

- Fixed bottom tabs being hidden by Samsung/Android system navigation.
- Added safe-area handling to the root app and bottom sheets.
- Added a shared communication inbox with unread and action badges.
- Added realtime notifications for shared calendar, care, medical and membership changes.
- Added cross-parent item requests with add-to-handover and unavailable responses.
- Added request history and activity-log records.
