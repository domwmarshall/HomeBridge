# HomeBridge v0.7.0 — Calendar and Shared Planning

## Included

- Proper Monday-first month calendar with month navigation and Today shortcut.
- Dad/Mum overnight overlay on every calendar day.
- Automatic weekly handover markers based on the normal alternating schedule.
- Month and upcoming-agenda views with filters.
- Add, edit, delete and acknowledge shared events.
- School, handover, party, trip, medical, holiday and reminder event types.
- All-day events, end date/time, RSVP deadlines, location, responsible parent and notes.
- Private invitation/letter/event photographs.
- Link tracked belongings to an event and warn when they are at the wrong home.
- Add required belongings directly to the next handover checklist.
- Temporary care changes and holiday overrides without destroying the normal schedule.
- Edit the normal alternating schedule, anchor handover date and collection point.
- Fixed one-time parent invitation hashing, pending invitation display and revocation.
- Realtime refresh for care rules, overrides, members and invitations.
- Pull-to-refresh and cached shared records.

## Database requirement

Run `supabase/v0.7.0_patch.sql` once before installing the APK.
